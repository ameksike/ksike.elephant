* Ensamblado en programación de sistemas informáticos o software de programación un ensamblado o assembly, es una colección de uno o más archivos o ficheros, agrupados juntos para formar una unidad lógica.
* Componente de Software, un elemento de un sistema software que ofrece un conjunto de servicios, o funcionalidades, a través de interfaces definidas.
* module o módulo: es una parte autónoma de un programa de ordenador, un módulo de núcleo es aquel que extiende las funcionalidades del núcleo de un sistema operativo.
* plugin o complemento: plug-in (del inglés «[un] enchufable o inserción»), add-on («añadido»), conector o extensión.
es una aplicación que se relaciona con otra para aportarle una función nueva y generalmente muy específica. Esta aplicación adicional es ejecutada por la aplicación principal e interactúan por medio de la API. 
* library: una biblioteca es un conjunto de subprogramas utilizados para desarrollar software. 
* toolkit (conjunto, equipo, maletín, caja, estuche, juego (kit) de herramientas).
